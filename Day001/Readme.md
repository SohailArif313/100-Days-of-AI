# Day 1 – Starting My 100 Days of AI Journey

I officially started my 100 Days of AI challenge today. I felt unsure at first, but once the environment was set up and the code ran, the momentum took over.

---

## What I Learned & Practiced

Today was about solidifying Python basics: inputs, typecasting with `int()` and `float()`, and clean f-strings. 

I built three basic scripts to test myself:
* Temperature converter
* Variable-swapping program
* Volume calculator using `math.pi`

---

## Key Takeaway

**Never tackle a whole problem at once. Break it down into small, logical pieces, and the code becomes easy.**
