# 🧠 Day 1 – Starting My 100 Days of AI Journey

Today I officially started my 100 Days of AI challenge. Honestly, I felt a bit unsure in the beginning, but once I set everything up and started coding, it actually felt good.

---

## 💡 What I learned today

Today was all about the basics of Python. Simple things, but very important.

I mainly focused on:

- Understanding how `input()` and `print()` work properly  
- Converting user input into numbers using `int()` and `float()`  
- Learning how **f-strings** make output cleaner and easier to read  
- Realizing that coding is more about logic than remembering syntax

---

## 🛠 What I practiced

I tried a few small programs to test myself:

- A basic temperature converter  
- Swapping two numbers without shortcuts  
- A simple volume calculation using `math.pi`

Each problem helped me understand how to break logic into small steps.

---

## 💭 What I realized

The biggest thing I understood today is:

**If I try to code everything at once, I get confused. But if I break the problem into small parts, it becomes easy.**

Feeling a bit more confident now. It’s just Day 1, but I can already see improvement.
