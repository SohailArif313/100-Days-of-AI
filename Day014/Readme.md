# Day 14 - Advanced NumPy and Vector Operations

This folder contains my solutions for the Day 14 tasks, focusing on data cleaning, advanced array indexing, and matrix shape rules in NumPy.

## Problems Solved

* **Nearest Value Finder:** Found the closest number in an array to any target value by calculating absolute differences.
* **Multiples Filter:** Cleaned arrays by automatically replacing all numbers that divide evenly by 3 or 5 with 0.
* **Fancy Indexing:** Target-selected specific positions inside an array using a list of index coordinates to change their values.
* **Missing Data Fix:** Tracked down missing values (NaN) inside an array and replaced them cleanly with 0 or standard averages.
* **Cauchy Matrix:** Created a custom Cauchy math grid using subtraction, division, and automatic shape alignment rules.
* **Tanh Curve Plotting:** Plotted the Tanh math function using matplotlib to visually see how deep learning networks change input values on a graph.
* **Shape Compatibility Checker:** Wrote a function from scratch to check if two different array shapes can match up by comparing their sizes from right to left.

## Takeaways

Today's practice showed me how to clean dirty datasets and handle shape mismatch issues. Learning how to filter elements quickly and handle missing data helps you prepare raw data perfectly before passing it into machine learning models.
