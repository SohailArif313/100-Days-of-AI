# The Data Wrangling Playbook: From Burritos to Bowlers 🌯🏏🌸

Welcome! This repository is a place where I practice working with real-world datasets and turn raw data into clean and useful insights.

Instead of using only clean tutorial datasets, I wanted to work with data that feels more realistic and helps me improve my problem-solving skills. This project includes three different types of datasets: fast-food sales, cricket analytics, and the Iris flower dataset

---

## 🚀 Why This Project Exists

🚀 Why This Project?

Real-world datasets are not always clean. They often contain missing values, different formats, and data that needs cleaning.

This project is my hands-on practice for solving these problems using (**Pandas, NumPy, Matplotlib, and Seaborn**). 

---

## 📂 The Core Challenges & Solutions

## Projects Included

### 1. Chipotle Sales Analysis
- Combined quarterly sales data into a single dataset.
- Cleaned price values by removing currency symbols.
- Analyzed the most sold items, top revenue items, and average order value.

### 2. IPL Cricket Analytics
- Calculated batting and bowling statistics.
- Found Purple Cap winners.
- Analyzed death-over bowling performance and player records.

### 3. Iris Data Analysis
- Merged multiple Iris datasets into one complete dataset.
- Compared flower measurements across different species.
- Created simple visualizations using Matplotlib.
