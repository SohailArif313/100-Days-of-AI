# Day 17 - Mastering Pandas DataFrames

This folder contains my practice files for Day 17, focusing on structural DataFrame tasks, missing value identification, row/column filters, duplicate cleaning, and working with multi-column sports datasets.

## Problems Solved

* **DataFrame Setup & Info:** Built custom labelled dataframes and extracted key metrics like data shape, descriptions, and alternate records.
* **Row/Column Slicing:** Selected specific index positions (`iloc`) and extracted matching subsets using clean relational filtering.
* **Data Imputation:** Located missing `NaN` values across rows and replaced them accurately using column-specific mode calculations.
* **Data Cleaning:** Tracked unique entry frequencies, aggregated specific conditional criteria, and dropped duplicated entries permanently.
* **Dataset Standardization:** Imported real-world IPL match histories and systematically updated historical franchise names to match modern titles.
* **Aggregate Visualization:** Combined individual team column instances to build a clean distribution bar chart highlighting the top 5 most active teams.

## Takeaways

Working with multi-column tables highlights the true power of Pandas. Learning how to clean missing parameters, safely adjust categorical values, and combine independent row dimensions provides a strong foundation for handling messy engineering data.
