# Day 16 - Introduction to Pandas Series

This folder contains my solutions for the Day 16 tasks, focusing on the basics of Pandas Series, simple array math, data loading, and basic graph plotting.

## Problems Solved

* **Series Creation:** Built empty and structured data Series using custom indices.
* **Series Mathematics:** Performed basic operations (addition, subtraction, multiplication, and division) directly between matching data rows.
* **CSV Data Loading:** Imported real datasets from CSV files (like player runs and store items) into Pandas arrays.
* **Percentile Check:** Calculated specific data cut-offs (30th and 60th percentiles) to analyze value distributions.
* **Range Filtering:** Used conditional math filters to locate and count items within a target price bracket (1000 to 2000).
* **Distribution Plotting:** Generated and saved a visual histogram graph to view item frequencies across different pricing segments.

## Takeaways

Today's practice showed me how Pandas builds on top of NumPy to make data cleaning much more human-readable. Working with labelled rows and loading external files opens up real capabilities for business data tracking.
