# 🏛️ Day 8 – Advanced Object-Oriented Programming (OOP)

Day 8 focuses on mastering deeper OOP concepts in Python, moving from basic classes to handling real-world tracking, dates, and object relationships.

## Problems Solved So Far

* **Instance Counting:** Using class variables to track exactly how many objects are created.
* **Class Relationships:** Creating a `Deck` class that manages an internal list of `Card` objects with full shuffle and deal mechanics.
* **Strict Encapsulation:** Building a `Rectangle` class using private properties and custom getters/setters to protect state.
* **Datetime Integration:** Designing a product system using the `datetime` module to calculate precise expiration timelines.

##  **Key Takeaway**

>  ***Writing clean classes is about more than just data storage. Handling relationships between multiple classes and protecting data using private fields builds robust systems that do not break as projects grow.***
