# Day 10 - File Handling and Recursion

This folder contains my complete solutions for the Day 10 tasks, covering intermediate file operations and recursive algorithm design in Python.

## Problems Solved

* **Efficient Line Reading:** Created a function to read and return only the final line of a text file efficiently without loading the whole file into RAM.
* **Vowel Counter:** Built a script to parse text documents and count the frequency of each vowel using a dictionary.
* **File Data Processing:** Read double-column numeric data from a text file, processed the numbers, and wrote the calculated totals into a new tab-separated file.
* **Line Reversal:** Developed a script to reverse strings line-by-line while keeping the newline formatting intact.
* **Object Serialization:** Used regular expressions to clean text and applied the pickle module to save and load serialized data structures.
* **Recursive String Length:** Calculated the length of a string using pure recursion instead of loops or the built-in len function.
* **Recursive GCD:** Implemented the Euclidean algorithm to find the Greatest Common Divisor of two numbers without using loops.
* **String Edit Distance:** Coded a recursive function to find the minimum edit distance between two words based on insertion, deletion, and substitution costs.
* **Run-Length Encoding:** Designed a purely recursive approach to compress consecutive repeating characters into a count list.
* **Decimal to Binary:** Created a recursive function that converts base-10 integers into standard binary strings.

## Takeaways

Today was about mastering data efficiency and programmatic logic. The first half showed how to handle files safely on disk without wasting memory, while the second half helped build a solid mental model for solving complex, nested problems by finding clean base cases and recursive steps.
